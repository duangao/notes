# 一维情况下分布变换

已知 X 的概率密度分布为$$p_x(x)$$, CDF为$$P_x(x)$$

假设我们可以对X进行采样，而要进行采样的目标变量为Y，其pdf和cdf记作$$p_y(y)$$ $$P_y(y)$$

而x和y之间满足函数关系：
$$Y = y(X) $$， 是一个一一映射。

那么，$$P_y(y) = P_x(x)$$成立，
故 $$ y = P_y ^{-1}(P_x(x)) $$


同时，可求出pdf的关系：

$$p_y(y) = |\frac{dy}{dx}|^{-1} p_x(x)$$

# 多维情况
$$Y = y(X)$$, y是双射

$$p_y(y) = p_y(y(x)) = \frac{p_x(x)}{|J_y(x)|}$$

$$J_y(x)$$为其雅可比矩阵
